Making Inline Audio Easy and Seamless

If you want to change how a playing clip is styled, you can implement your own progress callback. The callback gets two arguments, the DOM element which was clicked to play a clip, and an integer percentage of the amount of the clip which has been played at the time it was called. You tell Soundcite about your alternative callback using the SOUNDCITE_CONFIG variable somewhere in a <script> tag on your page. This example uses a different color scheme. 

   var SOUNDCITE_CONFIG = {
         update_playing_element: function(el, percentage) {
             $(el).css({
                 'background' : '-webkit-linear-gradient(left, white, #c34528 ' + percentage + '%, white)'
             });
         }
     }
